# PhysioTrial

**PhysioTrial** é um software para gerenciamento de escalas de sessões terapêuticas em pesquisas clínicas, integrando heurísticas de agendamento com uma interface simples. A ferramenta permite organizar pacientes, monitores e salas, lidar com faltas e realocações e estimar o tempo total necessário para concluir um protocolo de atendimento. Embora tenha sido desenvolvido originalmente para estudos em fisioterapia, sua arquitetura foi pensada para extensões em outras áreas de aplicação.

## Principais recursos

- Geração automática de agendas de sessões a partir das regras do estudo, restrições de disponibilidade e intervalos máximos permitidos.  
- Realocação inteligente em caso de faltas ou mudanças na rotina do paciente, preservando a lógica do protocolo.  
- Interface interativa que facilita o uso por pesquisadores sem experiência em programação.  
- Ferramentas para estimar datas prováveis de conclusão do estudo e visualizar o andamento do agendamento.  
- Módulos internos organizados de modo extensível, permitindo ajustes e customizações para diferentes linhas de pesquisa.
